# An-Exploration-of-Principal-Component-Analysis-and-Its-Applications
NOTE:
This program runs in conjunction with a seperate .mat data file. It is important to ensure that the correct .mat file is downloaded and it is loaded into the code otherwise the program will not run.

INSTRUCTIONS:
1. Create a .mat datafile using Data_Assembly.m, or use the provided sample file
2. import the file into PCA_Script.m
3. The program is now in a ready state to run, preform Principal Component Analysis, as well as graph the dataset before and after reduction.
